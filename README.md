# Unidad 0, transición a Java equipo 1

## Integrantes
Juan Baldenegro, Trinidad Bunge, Romina Cantoni, Maximiliano López y Paulina Silveira.

## Objetivo del repositorio
- Comparar la sintaxis del lenguaje Java con la de otros lenguajes utilizados en cursos previos (C#, Python).
- Utilizar correctamente los tipos primitivos de datos evaluando el uso de estos dependiendo de cada caso de aplicación.
- Desarrollar programas en Java empleando estructuras de control selectivas e iterativas, reconociendo qué variantes aplicar según la situación que corresponda; 
- Utilizar correctamente distintas formas de bucles y cortes de control y conocer las ventajas y desventajas de cada uno. 
- Discutir los conceptos básicos de instancias de clase y la utilización de objetos en lenguaje JAVA de acuerdo al paradigma POO.
- Operar efectivamente el entorno de desarrollo integrado NetBeans para la construcción y desarrollo de programas sencillos en Java.
- Utilizar buenas fuentes de información para profundizar en el aprendizaje del lenguaje Java.
- Construir programa(s) java que utilicen arrays, recorriendo los mismos para mostrar sus elementos, asignando elementos individuales a otras variables acordes, etc..
- Programar tareas básicas de manipulación de strings en JAVA
- Construir programas Java que permitan leer y escribir archivos de texto
